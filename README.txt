Untuk menjalankan aplikasi dibutuhkan pygame. Cara menginstallnya
adalah :
1. Buka command prompt
2. Ketik

python -m pip install -U pygame --user

3. Untuk mengecek keberhasilan instalasi Ketik

python -m pygame.examples.aliens
